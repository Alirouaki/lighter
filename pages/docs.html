| Setup Lighter

The installation of lighter is very simple, you can install it as a file or import it as a flavor with cdn.

Using cdn

Using this CDN on the head section in your file HTML.

 
                      
<link rel="stylesheet" type="text/css" media="screen" href="https://gitcdn.xyz/repo/amine1107/lighter/master/lighter.min.css" />
Download the minified file

you need only a file to start your work, the minified file is too light, only has 10KB.


Download
Clone Lighter using Git

You can Also clone Lightx using Git

                       
$git clone https://github.com/amine1107/lighter.git
Using Npm

You can install Lighter as a npm package

                       
$npm i lighter.css

Lighter needs jquery to create a responsive menu, we'll see later how to create it

| Row

Row property specifies a grid item's size and location in a grid layout, is a part of container, and contains all colmuns.


<div class="container">
  <div class="row">
  <!-- This contain colmuns -->
  </div>
</div>

Code :

    <a href="#" class="toggle">
        <h5>|||</h5>

    <div class="link-left">

  $(".toggle").parent().toggleClass('active');
});
