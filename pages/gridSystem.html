# Containers

the container helps you create a structured and clean page, Lighter uses a width of 90% to give you a very good layout.

Container

row

Colmun

Colmun

Colmun

Code:

<div class="container">
  <!-- This contain a row and colmuns -->
</div>

You can also use the container fluid, large then container.

Code:

<div class="container-fluid">
  <!-- This contain a row and colmuns -->
</div>

Depending on the width of the viewport that the webpage is being viewed on, the .container class gives its div a specific fixed width. These lines don't exist in any form for .container-fluid, so its width changes every time the viewport width changes.

# Rows

Row property specifies a grid item's size and location in a grid layout, is a part of container, and contains all colmuns.

Code:

<div class="container">
  <div class="row">
  <!-- This contain colmuns -->
  </div>
</div>
