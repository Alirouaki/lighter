Responsive web design (RWD) is an approach to web design which makes web pages render well on a variety of devices and window or screen sizes. Recent work also considers the viewer proximity as part of the viewing context as an extension for RWD. Content, design and performance are necessary across all devices to ensure usability and satisfaction.

1 - Getting Started.

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

a - Download Lighter.

the installation of Lighter is very simple, you can get a copy by downloading the minified file : : Download

b -Using CDN

In a very simple way, you can also import the default flavor of Lightx using the CDN.


<link rel="stylesheet" type="text/css" media="screen" href="https://gitcdn.xyz/repo/amine1107/lighter/master/lighter.min.css" />
c - Using Git.

$ git clone https://github.com/amine1107/lighter.git
c - Using Npm.

$ npm i lighter.css

2 - Running the tests.

You can simply add Lightx to your file Html.

Sass.

Lighter is built with simplified Sass, Sass is the most mature, stable, and powerful professional grade CSS extension language in the world.

Structure of Lighter Sass :

|---- scss
   |---- src
      |---- components
          |---- accordion.scss
          |---- block.scss (Blockquotes)
          |---- buttons.scss
          |---- cards.scss
          |---- code.scss
          |---- forms.scss
          |---- navbar.scss
          |---- progressBar.scss
          |---- table.scss
      |---- core
          |---- base.scss 
          |---- reset.scss (Normalize.css)
          |---- variables.scss 
      |---- layout
          |---- grids.scss 
      |---- utilities
          |---- helpers.scss
          |---- image.scss
          |---- list.scss
          |---- spacing.scss
          |---- typo.scss (Typography) 
      |---- lighter.scss
