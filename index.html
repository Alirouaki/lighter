1 - Getting Started.

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

a - Download Lightx.

the installation of Lightx is very simple, you can get a copy by downloading the minified file : : Download

b -Using CDN

In a very simple way, you can also import the default flavor of Lightx using the CDN.

 
<link rel="stylesheet" type="text/css" media="screen" href="https://gitcdn.xyz/repo/amine1107/lightx/master/lightx.min.css" />
c - Using Git.
$git clone https://github.com/amine1107/lightx.git
| You can later download Lightx with npm and bower.

Sass.

Lightx is built with Sass, Sass is the most mature, stable, and powerful professional grade CSS extension language in the world.

Structure of Lightx Sass :

|---- scss
   |---- src
      |---- components
          |---- block.scss (Blockquotes)
          |---- buttons.scss
          |---- cards.scss
          |---- code.scss
          |---- forms.scss
          |---- navbar.scss
          |---- progressBar.scss
          |---- table.scss
      |---- core
          |---- base.scss 
          |---- reset.scss (Normalize.css)
          |---- variables.scss 
      |---- layout
          |---- grids.scss 
      |---- utilities
          |---- helpers.scss
          |---- image.scss
          |---- list.scss
          |---- spacing.scss
          |---- typo.scss (Typography) 
      |---- lightx.scss
