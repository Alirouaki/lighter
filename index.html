Lightweight, Flexible, and Modern Css framework

Lightx is a flexible Css micro framework,Based on the visual aspect, the framework offers you a clean and easy work. Enjoy !.

Features

  • Lightweight 10KB
  • Modern
  • 12 colmun Grids
  • Responsive
  • No Javascript
  • Built with Sass

PS: This is the Beta version of Lightx.

Installation

Using Git

Clone Lightx using git.

$git clone https://github.com/amine1107/lightx.git
| you can later download Lightx with npm and bower

Running the tests

You can simply add Lightx to your file Html.

‹link rel="stylesheet" type="text/css" href="lightx.min.css"›
| You have now everything you want, so Happy Coding !
